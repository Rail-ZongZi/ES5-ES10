- 使用解决问题
  - 1： 使用 `ES6-ES10` 新特性，但是有兼容问题
  - 2： 使用 `Less`、`Sass`、`PostCSS`增强**CSS**编码性，运行环境不支持
  - 3： 使用**模块化**提高项目可维护性，运行环境不支持
  - 4： 部署项目上线需要**手动**压缩资源文件
  - 5： 部署过程需要**手动**上传代码到服务器
  - 6： 多人协作开发一个项目时，无法硬性保持**代码风格**
  - 7： 完全依赖后端服务**接口**支持

> 安装`es10-cli`文件

- `npm install es10-cli -g`

- `es10-cli create xxx`
